# Bitcoin Market Sentiment vs Trader Performance

This project analyzes whether Bitcoin market sentiment (Fear/Greed Index) 
correlates with trader profitability, win rate, and trading volume, 
using historical trade data from Hyperliquid.

## Key Findings
- Average profit per trade is highest during Extreme Greed and Fear periods
- Trading volume peaks during Fear days
- Win rate is highest during Extreme Greed (~89%)

## Files
- `assignment.ipynb` — full analysis notebook
- `fear_greed_index.csv` — sentiment data
- `historical_data.csv` — trading data
